# 두근두근 세일럼

세명대 연기예술학과 `베티와 마녀들` 팀의 공연 홍보/몰입용 캐릭터 대화 게임입니다.

## 포함 기능

- 아비게일, 존 프락터, 엘리자베스 프락터 3인 스토리모드
- 챕터 선택지, 호감도, 배드/노멀/굿 엔딩
- 전 인물 자유대화 모드
- 보고서 요구사항 기반의 현대 직업, MBTI, 숨은 감정, 대화 성향
- 논문/비평 방향을 반영한 인물 해석
- 외부 이미지 없이 바로 보이는 SVG 일러스트
- 브라우저 `localStorage` 저장

## 실행

`index.html`을 브라우저로 열면 바로 실행됩니다.

GitHub Pages에 올릴 때는 이 폴더 전체를 저장소에 올린 뒤 Pages의 배포 경로를 루트 또는 `/docs`로 지정하면 됩니다.
